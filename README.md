# Digital Payments Onboarding Demo

Live Demo: (https://698f1bc16f05ae4e039dda45--tubular-brigadeiros-43faa5.netlify.app/)

## Problem Statement
Many users drop off before completing their first transaction in payment apps.

## Objective
Improve first-time user onboarding and activation.

## Target Users
- New digital payment users
- Tier-2 and Tier-3 city users

## Solution
- Simplified onboarding flow
- Step-by-step guidance
- Clear call-to-action for first payment

## Metrics
- Activation Rate
- First Transaction Completion Rate

## Tech Stack
- HTML
- CSS
- JavaScript
